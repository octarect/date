# date

Reimplementation of the GNU date by Rust (for my studying).

- - -

## Usage

```shell
$ date
Fri Feb 17 12:04:40 JST 2017
```

Available options;

| option    | function                        |
|:---------:|:-------------------------------:|
| -u, --utc | print Coodinated Universal Time |
